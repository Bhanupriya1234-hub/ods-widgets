odsThemePicto
directive in module ods-widgets

Description

The odsThemePicto widget displays the pictogram of a theme based on the themes setting in ODSWidgetsConfig. This element can be styled (height, width, etc.), especially if the pictogram is vectorial (SVG).

Usage

as element:
<ods-theme-picto
       theme="{string}">
</ods-theme-picto>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
themestring

The label of the theme to display the pictogram of
