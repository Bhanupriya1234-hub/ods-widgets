odsPageRefresh
directive in module ods-widgets

Description

The odsPageRefresh widget can be used to periodically refresh the page.

Usage

as element:
<ods-page-refresh
       delay="{Number}">
</ods-page-refresh>
as attribute
<ANY ods-page-refresh
     delay="{Number}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
delayNumber

The number of milliseconds to wait before refreshing the page. The minimum value is 10000.

(default: 10000)
