odsRecordImage
directive in module ods-widgets

Description

The odsRecordImage displays an image from a record.

Usage

as element:
<ods-record-image
       context="{DatasetContext}"
       record="{Object}"
       field="{string}"
       domain-url="{string}">
</ods-record-image>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
contextDatasetContext

Dataset Context to use

recordObject

Record to take the image from

fieldstring

Field to use. By default, the first file field is used, but you can specify the field name if there is more than one field.

(default: none)

domainUrlstring

The base URL of the domain where the dataset can be found. By default, the current domain is used.

(default: none)
